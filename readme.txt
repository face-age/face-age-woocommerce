== Description ==

FaceAge A skin beauty and health care plugin that allows you to create interactive data visualizations for use with woocommerce. With the features you've come to expect, a skin data-driven beauty and healthcare platform that delivers a customized solution through detailed skin analysis. faceAge can be used in commercial and non-commercial projects.
Effortlessly assess your skin condition with unparalleled accuracy using just a single image.

If you use [WooCommerce](https://wordpress.org/plugins/woocommerce/), you can add this plugin to your site and your users can easily access the products that suit their skin by scanning their faces.

- **AI Facial Skin Analyzer.
- **Personalization Product Recommendations.
- **Personalization skincare routines.
- **Increased Average Order Value.

== Installation ==

1. install WooCommerce plugin
2. Visit `Plugins > Add New`
3. Search for `Face Age`
3. Install Face Age once it appears
4. Activate Face Age from your Plugins page.
5. Go to "after activation" below.


= Manual =

1. Upload the face-age-woocommerce folder to the /wp-content/plugins/ directory
2. Activate the Face Age plugin through the ‘Plugins’ menu in WordPress
2. Go to “after activation” below.

= After activation =

1. Go to the Face Age section from the WordPress menu
2. Enter your faceageId in the relevant section
3. Place the `[FaceAge type='skincare-analyzer']` and `[FaceAge-products]` shortcode in the section where you want the analyzer to be displayed (example: create a new page and place it in the desired section.)
4. Enter your faceageId in the relevant section
